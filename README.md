• Emart E-Commerce Application
This repository contains an E-commerce web application developed using Java and Spring Boot as part of the Advanced Java academic project and
The application simulates a basic online shopping platform with product management and user interaction features.


• About This Project
The Emart E-Commerce Application is designed to demonstrate core concepts of:
Spring Boot application development,MVC architecture,Database integration, Web dev using Java


• Key Features
Product listing and management
Add, update, and delete products
User interaction with product catalog
Database connectivity

• Tech Stack
Language: Java
Framework: Spring Boot
Architecture: MVC
Database: MySQL
IDE: Spring Tool Suite 4

• How to Run
1. Clone the Repository
git clone https://github.com/Sibasish45/Emart-EcommerceApp-.git
2. Open in Spring Tool Suite 4
Import → Existing Maven Project
Select the project folder
3. Configure Database
Update application.properties with your database credentials
Create the required database in MySQL 
4. Run the Application
Right-click project → Run As → Spring Boot App
Open browser http://localhost:8080
